# 📦 microservices-study1

## 🧭 Visão Geral

Este projeto é uma base de estudos sobre **arquitetura de microservices**, explorando a divisão de aplicações em serviços menores, independentes e escaláveis.

Funciona como um ambiente de aprendizado para entender como sistemas distribuídos são construídos na prática.

---

## 🔗 Relação com outros projetos

Este repositório é a base do ecossistema de estudos.

Ele serve como ponto de partida para experimentações que evoluem para projetos mais práticos como `microservice_conect`.

- `microservices-study1` → base teórica e experimental
- `microservice_conect` → evolução com integração entre serviços

---

## 🧠 Conceitos Estudados

- Separação de responsabilidades
- Arquitetura de microservices
- Comunicação entre serviços
- APIs REST
- Design de sistemas distribuídos

## ⚙️ Estrutura
